Windows
=======

This is repository for windows setting files.

vimedit was move in [new repository](https://github.com/rcmdnk/vimedit).

# Installation

Execute below commands in Cygwin:

    mkdir -p ~/tmp;cd ~/tmp
    git clone --recursive git@github.com:rcmdnk/scripts
    cd scripts
    ./install.sh
